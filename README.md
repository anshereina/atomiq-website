# AtomIQ - Nuclear Science Learning App

AtomIQ is a gamified educational application designed to make nuclear science accessible and fun for young learners. Built by IT students passionate about STEM education.

## Features

1. **Gamified Learning Path: The Mission** - Embark on an epic journey through interactive nuclear science lessons
2. **Myth Busters Module** - Challenge misconceptions about nuclear science
3. **Community** - Connect with fellow learners and share achievements
4. **Ask Schrodi** - AI-powered assistant to help with tricky concepts

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI Components
- React Router
- TanStack Query

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:8080`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
